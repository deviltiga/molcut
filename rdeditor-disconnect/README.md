# molcut

## rdEditor

A molecule editor based on RDKit and PySide6.

### Running the Editor

Run the script directly:

```bash
python rdeditor-disconnect/rdeditor/rdEditor.py
```

Or launch with a molecule file:

```bash
python rdeditor-disconnect/rdeditor/rdEditor.py molecule.mol
```

### Dependencies

- Python 3.x
- PySide6
- RDKit
- qdarktheme
